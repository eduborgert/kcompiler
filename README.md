# kcompiler
**Kcompiler** é um compilador para a linguagem "2020-01" da disciplina de compiladores, curso de Ciência da Computação da UNIVALI.

## Etapas/Status
- Lexico - ***ok***
- Sintático - *aguardando implementação*
- Semantico - *aguardando implementação*
